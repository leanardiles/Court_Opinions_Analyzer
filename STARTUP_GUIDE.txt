================================================================================
COURT OPINIONS ANALYZER - STARTUP GUIDE
================================================================================

This guide shows you how to start your development servers for both the 
backend (FastAPI) and frontend (React + Vite) applications.

================================================================================
PREREQUISITES
================================================================================

Make sure you have:
- Python 3.11+ installed
- Node.js 18+ installed
- Git Bash (or any terminal)
- PostgreSQL installed and running (for production use)

================================================================================
STARTING THE BACKEND (FastAPI)
================================================================================

STEP 1: Open Git Bash / Terminal

STEP 2: Navigate to the backend folder
--------
cd ~/Documents/Katz\ School/Capstone_Court_Opinions_Analyzer/backend

(Adjust the path based on where you cloned the repo)

STEP 3: Activate the virtual environment
--------
# On Mac/Linux:
source venv/bin/activate

# On Windows (Git Bash):
source venv/Scripts/activate

You should see (venv) appear at the start of your terminal prompt:
(venv) your-username@computer backend $

STEP 4: Start the FastAPI server
--------
uvicorn app.main:app --reload --port 8000

OR (if you have a main.py at the root):
python app/main.py

You should see:
INFO:     Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit)
INFO:     Started reloader process
INFO:     Started server process
INFO:     Waiting for application startup.
INFO:     Application startup complete.

STEP 5: Test it's working
--------
Open your browser and go to:
http://localhost:8000/docs

You should see the FastAPI auto-generated API documentation (Swagger UI).

STEP 6: Keep this terminal running
--------
Leave this terminal open while you're developing.
The server will auto-reload when you save Python files.

To STOP the backend:
Press Ctrl+C in the terminal

================================================================================
TROUBLESHOOTING
================================================================================

ISSUE: "command not found: uvicorn"
FIX: 
  1. Make sure venv is activated (you should see (venv) in prompt)
  2. Run: pip install uvicorn
  3. Try again

ISSUE: "Module not found" errors in backend
FIX:
  1. Activate venv: source venv/bin/activate
  2. Install dependencies: pip install -r requirements.txt
  3. Try again

ISSUE: "Cannot find module" errors in frontend
FIX:
  1. Make sure you're in the frontend folder
  2. Run: npm install
  3. Try again

ISSUE: Port 8000 or 5173 already in use
FIX:
  1. Find what's using the port:
     - Windows: netstat -ano | findstr :8000
     - Mac/Linux: lsof -i :8000
  2. Kill the process or use a different port:
     - Backend: uvicorn app.main:app --reload --port 8001
     - Frontend: npm run dev -- --port 5174

ISSUE: Tailwind styles not working
FIX:
  1. Make sure src/index.css has:
     @tailwind base;
     @tailwind components;
     @tailwind utilities;
  2. Run: npm install -D @tailwindcss/postcss
  3. Restart: npm run dev

================================================================================
STOPPING EVERYTHING
================================================================================

To stop development:
1. Go to Terminal 1 (backend) and press Ctrl+C
2. Go to Terminal 2 (frontend) and press Ctrl+C
3. Deactivate backend venv: deactivate
4. Close terminals
